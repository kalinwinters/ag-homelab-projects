# 🧠 HomeLab Projects

Repositorio personal donde documento y desarrollo distintos proyectos relacionados con:

* 🖥️ Sistemas y servidores
* 🌐 Redes
* 🔐 Ciberseguridad
* ⚙️ Infraestructura
* 🧪 Pruebas y experimentación técnica

---

## 🎯 Objetivo

Este repositorio tiene como finalidad:

* Centralizar proyectos personales en un solo lugar
* Documentar procesos técnicos y aprendizajes
* Experimentar con distintas tecnologías
* Construir un portafolio práctico en TI

---

## 🧩 Enfoque

Los proyectos dentro de este repositorio pueden variar en alcance y tecnología, incluyendo:

* Configuración de servidores
* Pruebas de rendimiento y estrés
* Automatización
* Laboratorios de red
* Experimentos técnicos

Cada proyecto puede tener su propia estructura, documentación y objetivos.

---

## 🚀 Tecnologías

Algunas de las tecnologías utilizadas o exploradas:

* Linux (Debian / Ubuntu Server)
* Nginx / Apache
* Herramientas de testing (k6, Apache Bench)
* Bash / scripting
* Docker
