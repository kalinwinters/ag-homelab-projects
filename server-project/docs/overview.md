# 🎯Objetivo del servidor

El objetivo de este servidor consistira en ser usado como ambiente de pruebas y entregar servicios dedicados.

---
## 🧠¿Para que sera usado?

El servidor pricipalmente sera usado como herramienta para distintas tareas en proyectos tales como:
<li>Pruebas de estres</li>
<li>Pruebas de carga</li>
<li>Escaneo de vulnerabilidades (al igual que testeos de ciberseguridad)</li>
<li>Monitoreo de peticiones</li>
<li>Monitoreo de recursos (web y servidor)</li>
<li>Alojamiento de webs</li>
<br>
tambien de distintas tareas que puedan requerir software dedicado que no se hagan mencion.

---
## ⭐Contexto

Se requiere crear un servidor con componentes desactualizados, esto con el fin de ahorrar recursos y darles un nuevo uso
a componentes los cuales su vida util ya a acabado.

No obstante, si las especificaciones del proyecto y sus requerimientos escalan con el tiempo, se pueden proprocionar aditamentos 
que puedan suplir la escalada de recursos y peticiones sean como:

<li>Mas RAM</li>
<li>Mas almacenamiento</li>
<li>Mejor bateria</li>
